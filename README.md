## BddGoogleSearch
  * When I go to the Google search page, and search for an item, i expect to see some reference to that item in the result summary.
  
 
## technologies used within the project: 
* Java 11.0.10
* Apache Maven 3.6.3
* Eclipse
* Selenium-java-3.141.59
* Chromedriver
* Jenkins
* Cucumber plugin
